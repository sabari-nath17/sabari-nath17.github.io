

# Minimalist Personal Bio Website

A clean, monochrome single-page personal bio site for a startup founder.

## Layout & Sections

1. **Hero Section**
   - Centered profile photo with grayscale filter and subtle rounded styling
   - Founder's name in large, bold typography
   - One-line tagline (e.g., "Founder & CEO at [Startup]")

2. **About Me Section**
   - Short bio paragraph (2-3 sentences placeholder text)
   - Clean divider between sections

3. **Social Links**
   - Minimal icon-based links (Twitter/X, LinkedIn, GitHub, Email)
   - Subtle hover effects (opacity or underline)

4. **Footer**
   - Simple copyright line

## Design
- **Colors**: Pure black, white, and grays only — monochrome palette
- **Typography**: Clean sans-serif, large headings, comfortable reading size for body
- **Spacing**: Generous whitespace for a premium, minimal feel
- **Responsive**: Single-column layout that looks great on mobile and desktop
- **Profile image**: Placeholder with CSS grayscale filter applied
